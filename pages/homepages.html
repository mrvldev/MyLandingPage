Learning Web Development
Alongside Python, I’ve also started learning HTML, CSS, and JavaScript, as well as Java, through The Odin Project.
It’s been a great way to understand how websites are built from the ground up — from simple static pages to interactive web applications.
The Odin Project provides a structured path that combines theory with hands-on projects, helping me practice real-world skills.
I’m currently working through the Foundations section, learning how to structure pages with HTML, style them with CSS, and add interactivity using JavaScript.
My goal is to build a solid understanding of front-end development, then move on to back-end technologies and full-stack projects. Each lesson brings me closer to creating my own complete and functional web experiences.
